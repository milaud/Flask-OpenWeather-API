Steps to run this project locally:

1. Clone this repo: git clone https://github.com/milaud/Flask-OpenWeather-API.git

2. Use a Python 2.7 environment.

3. Open project directory in terminal and run 'python miniproject3.py'.

4. You can enter cities in the form of CityName, CountryCode. Ex. Los Angeles, US or London, UK.

You can also see this project live on heroku: https://frozen-brushlands-88790.herokuapp.com/
